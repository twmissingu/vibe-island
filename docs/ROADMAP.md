## Phase 1: 宠物系统完善（进行中）

### 1.1 像素皮肤设计（5等级 × 8状态）

| 等级 | 名称 | 视觉风格 | 特效 |
|------|------|----------|------|
| L1 | 基础款 | 标准像素配色 | 无 |
| L2 | 辉光款 | 金色描边 | 轻微发光 |
| L3 | 金属款 | 银灰金属质感 | 高光反射 |
| L4 | 霓虹款 | 赛博朋克配色 | 霓虹光晕 |
| L5 | 王者款 | 皇冠装饰 | 粒子特效 |

8种状态动画：idle/thinking/coding/waiting/celebrating/error/compacting/sleeping

**工作量预估**：2-3天

### 1.2 缺失宠物动画（4种）

需补全：
- penguin（企鹅）- 摇摆走路、滑行
- robot（机器人）- 机械关节、LED眼
- ghost（幽灵）- 漂浮、透明效果
- dragon（小龙）- 翅膀扇动、喷火

**工作量预估**：3-4天

### 1.3 皮肤切换UI
- 设置面板添加皮肤选择器
- 网格展示已解锁皮肤
- 预览当前宠物不同皮肤

**工作量预估**：1天

## Phase 2: 成就系统

### 2.1 成就分类（30+）

| 类别 | 示例成就 | 触发条件 |
|------|----------|----------|
| 编码时长 | 编码新手(10h)、编码大师(100h)、传奇(1000h) | 累计时长 |
| 工具使用 | 多任务并行(3工具同开)、Claude死忠(50h) | 同时使用/单工具时长 |
| 宠物收集 | 初识伙伴(解锁3只)、动物园园长(全部解锁) | 解锁数量 |
| 连续记录 | 7天坚持、30天习惯 | 连续编码天数 |
| 特殊事件 | 深夜编码(凌晨2点)、周末加班 | 特定时间编码 |

**工作量预估**：2天

### 2.2 每日/每周挑战
- 随机生成3个日常挑战
- 完成奖励XP加成（1.5x / 2x）
- 挑战类型：编码时长目标、使用特定工具、完成特定状态切换次数

**工作量预估**：1-2天

### 2.3 本地排行榜
- 按工具分类（Claude/OpenCode/Codex时长排名）
- 总计编码时长排名
- 最近7天/30天排名

**工作量预估**：1天

## 开发优先级排序

| 优先级 | 功能 | 理由 |
|--------|------|------|
| P0 | 1.2 缺失宠物动画 | 补全产品完整性 |
| P0 | 1.1 像素皮肤设计 | 核心视觉体验 |
| P1 | 1.3 皮肤切换UI | 功能落地 |
| P1 | 2.1 成就系统 | 提升用户留存 |
| P2 | 2.2 每日挑战 | 增加用户粘性 |
| P2 | 2.3 本地排行榜 | 数据可视化 |

**总预估工期**：Phase 1 + Phase 2 = 11-14天（全职开发）
