## 数据模型对齐确认

### SessionState（AI 内部状态，7 种）

| 状态 | 说明 | 对应 IslandStatus |
|------|------|------------------|
| `idle` | 空闲，等待输入 | ⚪ `idle` |
| `thinking` | 思考中，处理 prompt | 🟢 `running` |
| `coding` | 编码中，调用工具 | 🟢 `running` |
| `waiting` | 等待用户审批 | 🟡 `waiting` |
| `completed` | 任务完成 | 🟢 `running` |
| `error` | 出错/中止 | 🔴 `error` |
| `compacting` | 上下文压缩中 | 🟠 `compacting` |

### IslandStatus（UI 显示状态，5 种）

| 状态 | 颜色 | 边框 | 宠物动画 | 声音 |
|------|------|------|---------|------|
| `idle` | ⚪ `#8E8E93` | 1px 灰色 | sleeping | 无 |
| `running` | 🟢 `#34C759` | 1.5px 绿色 | coding/thinking | 无 |
| `waiting` | 🟡 `#FFCC00` | 2px 黄色 | waiting（举手） | 叮声（2s 重复） |
| `error` | 🔴 `#FF3B30` | 2.5px 红色 | error（倒地） | 警告音（1 次） |
| `compacting` | 🟠 `#FF9500` | 1.5px 橙色 | compacting（背包） | 轻柔音（1 次） |
