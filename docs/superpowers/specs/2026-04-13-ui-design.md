# Vibe Island — UI/UX 设计文档 v1

> 设计目标：打造 macOS 上最美观、最实用的 AI 会话状态感知 HUD
> 设计理念：最小干扰、最大信息密度、像素级精致细节
> 设计日期：2026-04-13

---

## 1. 设计原则

### 1.1 核心设计哲学

**"氛围陪伴，而非打扰"**

Vibe Island 不是传统的监控工具，而是一个**有生命力的数字伴侣**。它在菜单栏上方安静地陪伴，只在重要时刻温柔地提醒你。

### 1.2 设计三要素

| 要素 | 说明 | 实现 |
|------|------|------|
| **感知** | 一眼识别当前状态 | 颜色编码 + 宠物动画 |
| **陪伴** | 有温度的交互体验 | 像素宠物 + 声音提醒 |
| **优雅** | 不打断工作流 | 最小占用空间 + 平滑动画 |

### 1.3 设计约束

- **空间限制**：收起态高度 32-40px，宽度 200-280px
- **视觉层级**：浮动在菜单栏上方，statusBar + 1 级别
- **信息密度**：收起态 3 秒内理解状态，展开态 10 秒内获取详情
- **动画时长**：展开/收起 ≤ 400ms，状态切换 ≤ 250ms

| 区域 | 宽度 | 内容 | 视觉 |

### 3.3 展开/收起动画

**动画曲线**：Spring 动画

```swift
.animation(
    .spring(
        response: 0.35,
        dampingFraction: 0.7,
        blendDuration: 0.2
    ),
    value: isExpanded
)
```

**动画流程**：

1. **点击收起态** (0ms)
2. **宽度扩展** (0-150ms)：240px → 320px
3. **高度扩展** (50-250ms)：36px → 240px
4. **内容淡入** (150-300ms)：透明度 0 → 1
5. **完成** (350ms)

**收起流程反向**，内容先淡出，再收缩。

### 4.3 渲染技术

```swift
// SwiftUI Canvas 渲染像素宠物
Canvas { context, size in
    let scale: CGFloat = 2.0  // 2x 缩放
    let pixelSize = size.width / 14.0  // 14 列
    
    for row in 0..<10 {
        for col in 0..<14 {
            let hexIndex = frameData[row][col]
            guard let color = palette[hexIndex] else { continue }
            
            let rect = CGRect(
                x: CGFloat(col) * pixelSize,
                y: CGFloat(row) * pixelSize,
                width: pixelSize,
                height: pixelSize
            )
            
            context.fill(
                Path(rect),
                with: .color(Color(hex: color))
            )
        }
    }
}
.antialiased(false)  // 保持像素锐利
```

**关键渲染参数**：
- `.antialiased(false)`：禁用抗锯齿，保持像素边缘锐利
- `.interpolation(.none)`：无插值，像素风格
- `scale: 2.0`：2x 缩放，14×10 → 28×20px

## 11. 设计总结

**Vibe Island 的 UI 设计核心**：

1. **美观性**：玻璃态 + 状态色渐变 + 像素宠物，打造独一无二的视觉风格
2. **实用性**：3 秒理解状态，10 秒获取详情，最小干扰工作流
3. **情感化**：宠物进化、声音陪伴、颜色呼吸，让监控变成陪伴
4. **一致性**：4px 网格间距、SF 字体家族、spring 动画，统一设计语言

**与竞品的视觉差异**：

| 竞品 | 视觉风格 | 我们的优势 |
|------|---------|-----------|
| claude-buddy | Electron 硬边框 | 原生玻璃态，更精致 |
| cc-status-bar | 纯文本状态 | 像素宠物 + 颜色动画 |
| cctop | 菜单栏列表 | 灵动岛浮动，更直观 |

**最终目标**：让用户爱上等待，享受与 AI 协作的每一刻。🏝️
