### 技术架构

| 架构特性 | 实现方式 |
|---------|---------|
| **四级降级** | Plugin → SSE → File → Process（OpenCode 专属） |
| **并发安全** | flock 文件锁 + DispatchSource 防抖动 |
| **状态聚合** | 多工具优先级排序（审批 > 错误 > 运行 > 空闲） |
| **数据持久化** | UserDefaults + JSON 文件 |
| **MVVM 模式** | @Observable + @MainActor |
